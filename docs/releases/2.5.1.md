title: Stability and Data Rights
summary: 补齐自主记忆运营的熔断、回滚、数据导出删除、规模化检索与 Shadow 评测闭环。
---

- 数据导出升级为 `life-memory-export/v4`，覆盖对话账本、工作案件、维护动作、证据封存、正式记忆摘要和 Graph Shadow 指标；不导出密码、向量正文或内部 Prompt。
- 新增完整账户删除和增强后的对话删除，清理数据库权威数据、派生索引、Workspace 镜像与安全范围内的媒体文件，同时保持跨用户隔离。
- 工作 Agent 新增 `active / shadow / paused_automatically / paused_manually / recovering` 维护状态、质量熔断和恢复验证；高风险维护暂停时不影响正常对话和证据保留。
- 自动合并、替代与过期维护提供 30 天可逆窗口，回滚恢复记忆版本、来源关系、向量投影和对话 Agent 的正式记忆 Brief。
- 去重检索改为按用户、状态、类型、敏感度和可见范围分区的有界近邻比较；10,000 条记忆验收中每批最多比较 20 x 20 个候选。
- 移除旧 MemoryRewriter、人工 hygiene 应用和手工合并入口；正式记忆写入继续由工作 Agent 的统一治理服务持有。
- 管理后台收敛为 5 个一级入口，并在 Agent Runtime 与设置页提供维护审计、暂停恢复、回滚、导出和删除能力。
- Graphiti 保持 Shadow-only：只记录不含正文的召回差异、来源覆盖、延迟和零 Token 指标，不获得正式记忆写权限。
