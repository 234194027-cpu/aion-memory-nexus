title: Autonomous Memory Operations
summary: Working Agent 统一接管记忆维护、证据封存与事件清理。
---

- 正式记忆的创建、修订、替代与维护继续由 Working Agent 的治理边界统一执行。
- 实时入口先做零 Token 门控；普通内容按来源最多 8 条、最长等待 15 分钟微批处理，明确记录、纠正和期限计划优先处理。
- 新增后台维护记录、EvidenceSeal 和 UserMemoryBrief。
- 每日 02:10 轻量维护、每周日 03:10 深度整理；每日每用户最多 24 次提取调用、4 次维护裁决调用。
- 已处理事件按 30/90/180 天策略自动删除或封存，正式记忆不因来源压缩而失效，并保留最小来源证明和 30 天可逆维护记录。
- 对话 Agent 读取 Working Agent 的正式记忆 Brief；取消旧的人工 rewrite/hygiene 写入旁路。
