# Aion Memory Nexus V3

此版本建立新的公开基线：以 `RawEvent → MemoryWorkCase → Evidence → Decision → CommittedMemory` 为权威数据主干，记忆用于支持复盘、模式分析、情景推演和用户自主决策。
